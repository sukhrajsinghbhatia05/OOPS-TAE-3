# Java Object-Oriented Programming (OOPS) - Teacher Assesment Exam 3 (Technical Profile Assesment)

This repository contains the Java source code for the Teacher Assesment Exam 3 (Technical Profile Assesment). Each program demonstrates core OOPS concepts such as Classes, Objects, Inheritance, Polymorphism, and Abstraction.

---

## 📝 List of Aims & Tasks

### 🏗️ Unit 1 & 2: Classes and Objects
- [x] **Aim 1:** Create a `Person` class with name/age attributes and a constructor.
- [x] **Aim 2:** Design a `Book` class and a `Library` class using `ArrayList` for management.
- [x] **Aim 3:** Define a `CARRENTAL` class with automated rent calculation based on car type.
- [x] **Aim 4:** Define a `Resort` class to compute stay charges with additional tax logic.
- [x] **Aim 5:** Implement an `Employee` class for net pay calculation (Basic + HRA + DA).
- [x] **Aim 6:** Create a `Rectangle` class to calculate area using parameterized constructors.

### 🧬 Inheritance, Polymorphism & Abstraction
- [x] **Aim 7:** Implement an abstract `Shape` class with concrete subclasses (`Circle`, `Rectangle`).
- [x] **Aim 8:** Define a `Filterable` interface implemented by `ImageProcessor` and `DataAnalyzer`.
- [x] **Aim 10:** Create a `Bank` interface for interest rate comparison (SBI, HDFC, ICICI).
- [x] **Aim 11:** Demonstrate **Hierarchical Inheritance** using a base `Shape` and multiple subclasses.
- [x] **Aim 12:** Design a payment processing system using an abstract `Payment` class and subclasses (`UPI`, `CreditCard`).
- [x] **Aim 13:** Evaluate performance using an abstract `Employee` class with **Runtime Polymorphism**.
- [x] **Aim 14:** Demonstrate **Multilevel Inheritance** using `Person` -> `Employee` -> `Manager`.
- [x] **Aim 15:** Implement a ride-booking application with dynamic fare calculation using a `Ride` array.
- [x] **Aim 16:** Model hospital staff responsibilities using method overriding.

### ⚙️ Keywords: Static, Final, and This
- [x] **Aim 9:** Demonstrate **Method Overloading** in a `Calculator` class (Compile-time polymorphism).
- [x] **Aim 17:** Track object creation count using a `static` variable and `static` method.
- [x] **Aim 18:** Implement a fixed interest rate system using the `final` keyword.
- [x] **Aim 19:** Manage a collection of book details using an **Array of Objects**.
- [x] **Aim 20:** Demonstrate object variable initialization using the `this` keyword.
